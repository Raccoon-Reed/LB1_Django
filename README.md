# 📰 Django Blog — Система авторизації та ролей

| Роль       | Читати | Коментувати | Створювати статті | Редагувати свої | Редагувати будь-які |
|------------|--------|-------------|-------------------|-----------------|---------------------|
| 👁️ Читач   | ✅     | ✅          | ❌                | ❌              | ❌                  |
| ✏️ Автор   | ✅     | ✅          | ✅                | ✅              | ❌                  |
| 🛡️ Модератор | ✅   | ✅          | ✅                | ✅              | ✅                  |
| 🔧 Адмін   | ✅     | ✅          | ✅                | ✅              | ✅                  |

## Встановлення та запуск

```bash
# 1. Встановити залежності
pip install django

# 2. Запустити налаштування
bash setup.sh

# 3. Створити суперкористувача (адміна)
python manage.py createsuperuser

# 4. Запустити сервер
python manage.py runserver
```

## Сторінки

- `/` — Список категорій
- `/category/<slug>/` — Статті категорії
- `/article/<id>/` — Стаття з коментарями
- `/article/create/` — Створити статтю (тільки автори+)
- `/article/<id>/edit/` — Редагувати (автор або адмін)
- `/article/<id>/delete/` — Видалити (автор або адмін)
- `/register/` — Реєстрація
- `/login/` — Вхід
- `/profile/` — Профіль користувача
- `/admin/` — Адмін-панель Django
